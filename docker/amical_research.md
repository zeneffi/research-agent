# Amical 調査レポート

**調査日**: 2026年1月22日

---

## 概要

**Amical**は、オープンソースのAI音声入力・ノート作成アプリケーション。macOS向けに開発されており、ローカルでもクラウドでも動作する。

| 項目 | 内容 |
|------|------|
| 公式サイト | https://amical.ai/ |
| 対応OS | macOS 12 (Monterey) 以降 |
| 現在のバージョン | 0.1.13 |
| ライセンス | オープンソース |
| 価格 | 無料 |

---

## インストール方法

Homebrewを使用して簡単にインストール可能：

```bash
brew install --cask amical
```

## LanguageMode（機能モード）について

Amicalには**4つのモード/機能**があり、初期設定時に使用したい機能を選択する。これが「LanguageMode」と呼ばれる設定項目。

### 1. Contextual Dictation（AI Dictation）

**コンテキスト対応の音声入力**

- **機能**: どのアプリでも使える、シームレスな音声入力（音声→テキスト）
- **特徴**:
  - キーボードの代わりに声で文章を入力
  - 現在使用中のアプリを自動認識
  - メール、チャット、ドキュメント編集など、あらゆる場面で使用可能
- **ユースケース**: 普段から文章をたくさん書く人向け

### 2. Note Taking

**スマートノート作成**

- **機能**: 音声から考えやアイデアを記録し、スマートな整形機能で構造化
- **特徴**:
  - 話した内容を自動で箇条書きに変換
  - 見出しを自動付与
  - 構造化されたメモを作成
- **ユースケース**:
  - ブレインストーミング
  - 会議のメモ
  - アイデアの書き留め
- **相性の良いアプリ**: Obsidian、Notion などの情報整理系アプリ

### 3. Meeting Transcriptions（Coming Soon）

**会議の文字起こし**

- **機能**: 会議や会話を高精度で録音・文字起こし
- **特徴**:
  - 複数人の会話に対応
  - 自動でテキスト化
- **ユースケース**:
  - Zoom会議
  - 対面での打ち合わせ
  - 電話でのやり取り
  - 議事録作成

> **注意**: 現在開発中（Coming Soon）

### 4. Voice Commands（Coming Soon）

**ボイスコマンド**

- **機能**: 音声でアプリをハンズフリー操作
- **特徴**: 自然な音声コマンドでタスクを実行

> **注意**: 現在開発中（Coming Soon）

## 初期設定

1. アプリ起動後、「Welcome to Amical」画面が表示される
2. 使用したい機能（LanguageMode）にチェックを入れる
3. 最低1つの機能を選択する必要がある
4. 設定は後からいつでも変更可能

**推奨**: 初めての場合は、上2つの機能（Contextual Dictation、Note Taking）にチェックを入れる

## 主な機能・特徴

| 機能 | 説明 |
|------|------|
| ローカル/クラウドモデル | ローカルでもクラウドでも動作 |
| カスタム語彙 | 専門用語などを登録可能 |
| 多言語サポート | 複数の言語・方言に対応 |
| カスタムショートカット | キーボードショートカットをカスタマイズ |
| BYO Model | 自分のAIモデルを使用可能 |
| BYO Key | 自分のAPIキーを使用可能 |
| MCP連携 | Model Context Protocol統合 |

## インサイト

1. **ローカル動作**: クラウドに依存せずローカルで動作するため、プライバシーを重視するユーザーに適している

2. **オープンソース**: 無料で使用でき、コードも公開されている

3. **macOS特化**: 現時点ではmacOS専用。Windows/Linuxユーザーは使用不可

4. **開発中機能**: Meeting TranscriptionsとVoice Commandsは「Coming Soon」のため、会議の文字起こしや音声コマンドが必要な場合は他のツールを検討

5. **Notion/Obsidianとの相性**: Note Taking機能は情報整理系アプリとの連携が良好
