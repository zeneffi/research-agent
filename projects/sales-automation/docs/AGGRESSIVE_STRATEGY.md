---

## 攻めの戦略

### 1. 量を増やす（短期）

**目標: 週100社 → 週300社**

- 毎日平日10時に自動実行（30社/日 × 5日 = 150社）
- 手動実行で+150社/週
- エリアローテーションで重複回避

### 2. 成功率を上げる（中期）

**目標: 平均成功率 50% → 70%**

- 失敗パターンを分析して対策
  - 電話番号形式エラー → 判定ロジック改善 ✅
  - 必須フィールド不足 → 姓名分割対応 ✅
  - フォーム検出失敗 → 検出ロジック強化
- CAPTCHAスキップではなく「送信試行」に変更 ✅

### 3. 返信率を上げる（中長期）

**目標: 返信率 1% → 3%**

- 営業文A/Bテスト
  - パターンA: 紹介パートナー訴求（現行）
  - パターンB: 課題解決訴求
  - パターンC: 実績訴求
- 送信時間帯の最適化（平日10時が開封率高い説）
- 相手企業への言及を強化（パーソナライズ）

### 4. 新規チャネル開拓（長期）

- LinkedIn InMail
- Twitter/X DM
- 比較ビズ/発注ナビの逆営業

## 業種拡大

### Phase 1（IT企業 → IT企業）
- システム開発会社へ紹介パートナー提案
- 同業種なので理解が早い

### Phase 2（IT企業 → 製造業DX）
- 製造業のDX需要を狙う
- 営業文を業種特化に

### Phase 3（IT企業 → 建設テック）
- 建設業のIT化需要
- 補助金と絡めた提案
