```
「LP制作」で広告出稿
   ↓

   ↓
月3万円で30-60クリック
   ↓
問い合わせ率1-2%
   ↓
月0-1件問い合わせ（費用対効果悪）
```

**⭕ 成功パターン**

```
「工務店 LP」でニッチ広告
   ↓
クリック単価50-150円
   ↓
月3万円で200-600クリック
   ↓
問い合わせ率3-5%（ターゲット明確）
   ↓
月6-30件問い合わせ（費用対効果良）
```

### 7.2 プラットフォームの力を借りる

**Coconalaの3大メリット**

1. **集客代行**: Coconala自体がSEOに強く、「LP制作」で検索流入がある
2. **信頼構築**: レビューシステムで初心者でも信頼獲得可能
3. **初期費用0円**: 売れてから手数料22%（リスク最小）

### 7.3 専門特化で差別化

**一般的なLP制作 vs 工務店専門LP制作**

| 項目 | 一般的なLP制作 | 工務店専門LP制作 |
|------|--------------|----------------|
| 競合数 | 数千件 | 20件（Coconala） |
| 単価 | 3-5万円 | 5-15万円（専門性プレミアム） |
| 成約率 | 1-2% | 5-10%（ターゲット明確） |
| リピート率 | 低い | 高い（業界内紹介） |

## 8. 工務店専門に特化すべき理由

### 8.1 圧倒的な需給ギャップ

```
┌────────────────────────────────────┐
│      需給ギャップの可視化           │
├────────────────────────────────────┤
│  工務店数: 40万社                   │
│  Coconala上のサービス: 20件         │
│  ギャップ: 1対20,000                │
│                                    │
│  → 競合がほぼいない市場             │
└────────────────────────────────────┘
```

### 8.2 工務店の広告予算は十分

| 年商 | 広告費（3-5%） | LP制作予算（推定） |
|------|---------------|------------------|
| 5,000万円 | 150-250万円/年 | 10-30万円/件 |
| 1億円 | 300-500万円/年 | 15-50万円/件 |
| 3億円 | 900-1,500万円/年 | 30-100万円/件 |

**インサイト**: 従業員10人未満でも年商5,000万-1億円規模は多く、LP制作に5-10万円を払える余裕がある

### 8.3 業界知識がバリアになる

**参入障壁**

- 建築用語（気密性、C値、UA値、耐震等級など）
- 業界慣習（見学会、OB顧客、紹介率など）
- 施主の心理（高額商品、長期検討、信頼重視）

**あなたの強み**

- 既に業界調査レポート作成済み
- 工務店の課題を理解している
- 専門用語を使いこなせる

→ 他のLP制作者は簡単に真似できない

## 12. まとめ: なぜこの戦略が機能するか

### 3つの成功要因

#### 1. ニッチ×ブルーオーシャン

```
┌─────────────────────────────────────┐
│  市場規模: 大（工務店40万社）          │
│  競合: 極小（Coconala20件）            │
│  → 独占可能                            │
└─────────────────────────────────────┘
```

#### 2. 低予算でも勝てる理由

- レッドオーシャンキーワードを避ける
- プラットフォーム（Coconala）の力を借りる
- 無料施策（SNS、SEO）を並行

#### 3. 専門性による高単価化

- 一般的なLP: 3-5万円
- 工務店専門LP: 5-15万円
- **プレミアム1.5-3倍**
