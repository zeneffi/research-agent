```
ご検討いただけますと幸いです。

[会社名/個人名]
[連絡先]
```

### パターンB: 具体的数字訴求型

```
件名: 【工務店専門】月5件の問い合わせを増やすLP制作のご提案

本文:

突然のご連絡失礼いたします。
工務店・住宅会社専門のWeb制作を行っております、[会社名/個人名]と申します。

突然ですが、1つ質問させてください。

「月に5件、問い合わせが増えたら、年間いくらの売上増になりますか？」

月5件の問い合わせ増 = 年間6件の成約増 = 1.2億円の売上増

この可能性を、1枚のLPで実現するお手伝いをしております。

貴社のホームページを拝見しました。
[具体的に良かった点を1つ記載]

しかし、LPがないことで機会損失が発生している可能性があります。

【私どもの強み】
・工務店業界に特化（専門用語・業界慣習を理解）
・成果にコミット（問い合わせ保証プランあり）
・短納期対応（最短2週間で公開可能）

まずは15分のオンライン面談で、
貴社のWeb集客の改善ポイントを無料でお伝えします。

ご都合の良い日時をお知らせいただければ幸いです。

[会社名/個人名]
[連絡先]
```

## 4. 使用時の注意事項

### NG事項
- [ ] 過度な営業感を出さない
- [ ] 「無料」を連発しない
- [ ] 長文になりすぎない（フォーム: 300字以内、DM: 200字以内推奨）
- [ ] 嘘や誇張した数字を使わない

### カスタマイズポイント
1. **会社名/サービス名** → 具体的に差し替え
2. **具体的に良かった点** → HPやInstagramを確認して1点記載
3. **季節感** → 見学会シーズンなど時期に合わせて調整
4. **実績数字** → 自身の実績に置き換え

### 送信タイミング
- 平日 10:00-12:00、14:00-16:00 が推奨
- 月曜・金曜は避ける（忙しい/週末モード）
- 火曜〜木曜がベスト
