LP制作サービス提供のための業界理解資料

## 1. 業界概要

### 工務店とは
地域に密着した住宅建設・リフォームを行う建設業者。ハウスメーカーと比較して、自由度の高い設計と地域特性に合わせた施工が特徴。

### 市場規模
| 指標 | 数値 |
|------|------|
| 住宅投資総額 | 約17兆円 |
| リフォーム市場 | 約7.3兆円 |
| 工務店数 | 約40万社以上 |
| 従業員10人未満 | 全体の約90% |

### 新設住宅着工戸数の推移
| 年度 | 総数 | 持家 | 前年比 |
|------|------|------|--------|
| 2023年 | 約81.9万戸 | 約22.5万戸 | -4.6% |
| 持家前年比 | - | - | -11.4% |

**注目**: 持家着工戸数が大幅減少 → 新築依存の工務店は厳しい状況

## 2. ビジネスモデル・収益構造

### 収益源（4つの柱）

```
┌─────────────────────────────────────────────────────┐
│                    工務店の収益源                      │
├─────────────┬─────────────┬─────────────┬───────────┤
│   新築工事   │  リフォーム  │   設計料    │ アフター   │
│             │    工事     │            │ メンテナンス│
│   粗利15-25% │  粗利25-35% │  工事費の   │  定期収入  │
│             │            │   5-10%    │           │
└─────────────┴─────────────┴─────────────┴───────────┘
```

### コスト構造
| 項目 | 割合 | 内容 |
|------|------|------|
| 材料費 | 30-40% | 木材、建材、設備機器 |
| 人件費 | 20-30% | 職人、設計士、営業 |
| 外注費 | 15-25% | 電気、設備、塗装等 |
| 広告宣伝費 | 3-5% | チラシ、Web、展示場 |
| 一般管理費 | 10-15% | 事務所、車両、保険 |

### 3つのビジネスモデル類型

#### ① 元請け型（自社完結）
- 顧客から直接受注
- 設計〜施工〜アフターまで一貫
- **利益率**: 高い（20-30%）
- **課題**: 集客力・営業力が必要

#### ② 下請け型
- ハウスメーカーや大手ゼネコンから受注
- 施工のみ担当
- **利益率**: 低い（5-10%）
- **課題**: 価格交渉力が弱い、依存リスク

#### ③ ハイブリッド型
- 元請けと下請けの併用
- リスク分散
- **利益率**: 中程度（15-20%）

## 4. 業界の課題（5つの重要課題）

### 課題①: 職人不足・高齢化
| 指標 | 現状 |
|------|------|
| 55歳以上の割合 | 36% |
| 29歳以下の割合 | 約10% |
| 10年後の予測 | 大量引退 |

**対策**:
- 若手採用強化（待遇改善、働き方改革）
- 多能工育成
- 女性・外国人材活用

### 課題②: 資材価格高騰
- ウッドショック（木材価格2倍以上）
- 半導体不足（設備機器納期遅延）
- 円安による輸入材高騰

**対策**:
- 原価管理の徹底
- 代替材の検討
- 価格転嫁の交渉力

### 課題③: 集客力の低下
- 人口減少・世帯減少
- 新築着工戸数の減少
- Web集客への対応遅れ

**対策**:
- **LP・Web広告の活用** ← ここにビジネスチャンス
- SNS活用
- 紹介・口コミ強化
- リフォーム事業への転換

### 課題④: 働き方改革対応
- 2024年問題（時間外労働上限規制）
- 週休2日制への対応
- 工期短縮の必要性

**対策**:
- 業務効率化（IT導入）
- 工程管理の改善
- プレカット・プレファブ活用

### 課題⑤: 後継者問題
- 経営者の高齢化
- 親族内承継の減少
- M&A・事業承継の需要増

## 6. LP制作サービス提案のポイント

### 工務店がLPを必要とする場面

| シーン | LP内容 | 目的 |
|--------|--------|------|
| 新築集客 | モデルハウス見学会 | 来場予約獲得 |
| リフォーム | 水回り/外壁塗装 | 問い合わせ獲得 |
| 採用 | 職人募集 | 応募獲得 |
| 完成見学会 | 期間限定イベント | 予約獲得 |

### 訴求すべきポイント

#### 新築LP
- 施工事例（写真・動画）
- 性能値（断熱・気密・耐震）
- 価格の透明性
- 地域での実績年数
- お客様の声

#### リフォームLP
- Before/After写真
- 工事期間の明示
- 価格帯の目安
- 補助金対応
- 保証内容

### 競合との差別化ポイント
1. **業界知識の深さ**: 専門用語・業界慣習の理解
2. **実績重視**: 工務店専門のLP事例
3. **成果保証**: 問い合わせ数の目標設定
4. **継続サポート**: LP公開後の改善提案

## 8. まとめ：LP制作の商機

### なぜ工務店にLPが必要か
1. **Web集客への対応遅れ**: 多くの工務店がまだチラシ・紹介依存
2. **競争激化**: 差別化のためのWeb発信が不可欠
3. **リソース不足**: 自社でWeb施策を行う余裕がない
4. **若年層へのアプローチ**: スマホ世代の施主獲得

### Coconala市場の状況
- 工務店向けLP制作サービス: **わずか20件**
- 工務店数: **40万社以上**
- 比率: **1:20,000** → 巨大な潜在市場
