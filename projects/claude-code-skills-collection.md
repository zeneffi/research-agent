## スキル作成のベストプラクティス

Anthropics公式READMEより抜粋:

### スキルとは
スキルは、Claudeの能力を大幅に向上させる指示・スクリプト・リソースのまとまりです。特定のタスクを再現性のある方法で完了する方法を教えます。

### スキル作成時のポイント
1. **ドメインを選ぶ** - 専門知識のある分野を特定
2. **ワークフローを定義** - 基本的なステップとロジックをマップ
3. **スキルを実装** - Claude Codeのスキル開発フレームワークを使用
4. **徹底的にテスト** - 様々なシナリオでスキルが動作することを確認
5. **明確にドキュメント化** - 包括的なドキュメントと例を提供

### ベストプラクティス
- **フォーカスを保つ** - 各スキルは1つの特定の問題を解決
- **エラーを適切に処理** - 適切なエラーハンドリングとバリデーションを含める
- **例を提供** - 使用例とエッジケースを含める
- **規約に従う** - 標準パターンとネーミング規約を使用
